## 🚀 部署说明

### 1. 数据库升级
```bash
# 方式1: 执行修复脚本
mysql -u root -p your_database < backend/database/fix_missing_columns.sql

# 方式2: 重新初始化（将 init.sql 导入）
mysql -u root -p your_database < backend/database/init.sql
```

### 2. 代码部署
```bash
# 编译打包
cd backend
mvn clean package -DskipTests

# 启动服务
java -jar target/enterprise-ad.jar
```

### 3. 运行测试
```bash
# 运行单元测试
mvn test

# 运行特定测试类
mvn test -Dtest=ConcurrencySafetyTest
```
